A) CSS Files and Structure

This template uses CDN to load vendors' css files, in case of fallback it uses css files contained in the css folder

Css files:

  • base
  • main
  • font-awesome
  • bootstrap

following CSS file are included for predefined color variations:

  • main-amber
  • main-black
  • main-green
  • main-orange
  • main-ping
  • main-purple
  • main-red
  • main-teal

Image variations:

  • main-image-dark
  • main-mage-light

main sass file layout:

main.scss is main scss layout which can be used to create variations or modify included css files.

Line:21-31 of main.scss file have predefined colors which can be used to configure template


        //color options
        $white: #ffffff;
        $amber: #FFCA28;
        $black: #000000;
        $blue: #42A5F5;
        $green: #66BB6A;
        $orange: #FFA726;
        $pink: #EC407A;
        $purple: #AB47BC;
        $red: #EF5350;
        $teal: #26A69A;

Line:33-35 have primary and secondary colors which can be changed to alter the color scheme of complete template. 

        $primary: $black;
        $light_primary: rgba($primary, 0.05);
        $secondary: $white;

 

B) Image Files

All the graphic elements for the entire template are contained at the "img" folder.

 

 

C) Contact Form

For addressing contact form too your email address, edit "contact.php" file and change email address on line 5.

 

E) JavaScript

BERTO includes external libraries from CDN, in case of fallback these files are linked at the end of document - after the "footer" tag of the document. If you need to modify the behavior of plugins or certain jquery element open the "js/main.js" file where you can alter the parameters of some of the js elements.

we have combined and minified all plugins being using in "js/plugins.js"

Vendor libraries are included in "js/vendor" folder
